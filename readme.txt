== Description ==

A WordPress plugin for displaying statistics about published post counts per month and per year.

= Features =

* Track post publishing statistics per month and per year
* Lightweight and fast performance
* Clean and intuitive interface

= Privacy =

The plugin does not collect any data. It uses the current WordPress posts table to compute its statistics

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/abnet-post-stats` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use WordPress to activate and view the two widgets: "Post Statistics per Month" and "Post Statistics per Year"
