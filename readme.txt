== Description ==

ABNet Post Stats is a comprehensive WordPress plugin that provides advanced post statistics, content organization, and professional-grade stylometric analysis for your WordPress site.

**Key Features:**

= Post Publishing Statistics =
* Monthly and yearly post count tracking
* Beautiful dashboard widgets with visual charts
* Historical publishing data analysis
* Customizable date ranges and filtering

= Content Pillars Management =
* Create custom content pillars based on categories
* Color-coded organization system
* Elegant category selection with search functionality
* Dedicated dashboard widgets for each pillar
* Track performance across content themes

= Advanced Stylometric Analysis =
* **LIX Readability Index** - Swedish readability measurement
* **Yule's K** - Vocabulary richness and lexical diversity
* **Negativity Analysis** - Sentiment and tone measurement
* **Punctuation Metrics** - Writing style indicators
* **Hapax Legomena (HTR)** - Unique word usage analysis
* **Entropy Measures** - Text complexity analysis

= Dashboard Integration =
* Multiple dashboard widgets for different metrics
* Widgets can be hidden/shown via Screen Options
* Responsive design with chart visualizations
* Color-customizable displays

= Professional Features =
* Academic-grade text analysis algorithms
* Export capabilities for research purposes
* Bulk operations for large content sets
* Developer-friendly with hooks and filters

**Perfect for:**

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/abnet-post-stats` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Settings → Content Pillars Definitions to set up your content pillars
4. View your statistics on the WordPress Dashboard

== Frequently Asked Questions ==

= What is stylometric analysis? =

Stylometric analysis is the computational study of writing style. It uses statistical methods to analyze linguistic features like vocabulary richness, sentence complexity, and writing patterns. This plugin implements several academic-grade stylometric measures.

= What are Content Pillars? =

Content Pillars are thematic groupings of your content based on categories. They help you organize and track performance across different content themes or topics on your site.

= How accurate are the readability scores? =

The plugin uses established academic formulas like the LIX readability index, which has been validated in linguistic research. Results are comparable to professional text analysis tools.

= Can I export the analysis data? =

Yes, the plugin provides various export options for research and analysis purposes.

= Does this affect my site's performance? =

The plugin is optimized for performance. Analysis is done in the background, and results are cached to minimize impact on your site's speed.
